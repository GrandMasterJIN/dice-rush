# Contributing

Thanks for your interest in Dice Rush.

This project is primarily a portfolio and learning artifact documenting AI-assisted product development. It is open sourced for reference and exploration rather than active feature development.

## You're welcome to

- Fork and experiment freely under the MIT license
- Open issues to report bugs or suggest improvements
- Submit pull requests for bug fixes or small improvements

## Before submitting a PR

- Run the test suite: `npm run test`
- Keep changes focused — one concern per PR
- Briefly describe what you changed and why

See the [README](README.md) for setup instructions.
